# DronePilotWeb 是什么？

- DronePilotWeb 是一个无人机高可视化 Web 仿真平台，采用 Vue 3 响应式框架与 Three.js 3D 引擎构建，为无人机自主识别与自动控制提供高保真度的训练环境。  
- 系统配备了高精度的无人机下视摄像头，可实时捕获无人机视角的图像数据。用户可以通过编写 JavaScript 代码来控制无人机的飞行，并通过底部摄像头实时获取场景图像。  
- 项目深度集成了 OpenCV.js 图像处理库，支持对摄像头数据进行实时分析和处理，让用户可以开发视觉导航、目标追踪等常用功能。这种基于视觉反馈的编程模式，高保真地模拟了例如无人机竞赛项目中的场景要求。
